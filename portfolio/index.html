Details

Arduino uno:

Arduino Uno is a microcontroller development board used to control sensors and electronic components in embedded systems. I used it in my Digital Parking System project, where it collected data from ultrasonic sensors and displayed available parking slots on an LCD screen. It served as the central controller for real-time hardware integration and automation.

ESP32:

ESP32 is an advanced microcontroller with built-in Wi-Fi and Bluetooth capabilities, widely used in IoT applications. I implemented ESP32 for wireless communication and remote data transmission in embedded systems. It offers high processing performance, low power consumption, and strong connectivity features.

Raspberry pi:

Raspberry Pi is a compact single-board computer that runs on a Linux-based operating system. I used it in my Real-Time Face Recognition System to execute Python-based backend logic using OpenCV and Dlib. It enabled real-time image processing and facial recognition on an edge computing device.
